/*** ARCHIVOS ***/
input               - Contiene los datos del problema, es decir, los ficheros *.set y *.const
output              - Contiene los datos de salida, esto es solo requerido si se hace uso del script gatherData.sh
GENE_exe            - Programa ejecutable que implementa los Algoritmos Genéticos
MEME_exe            - Programa ejecutable que implementa los Algoritmos Meméticos
gatherData_P2.sh    - Script utilizado para realizar las ejecuciones requeridas, los resultados los almacenan en output
seeds.txt           - Fichero que contiene las semillas utilizadas
